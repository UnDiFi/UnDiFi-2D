Purpose

Compute the faces of the mesh; one face normal (its NDIM cartesian components) is stored for each edge/face shared by two elements i.e. the normals of the inner (shared) faces are NOT duplicated; the pointer ICELFAC(i,j) is used to address the faces of a cell and know whether it is pointing inside or outside

Parameters
[in]ICELNODCell to node pointer: ICELNOD(i,j) gives the global node number of the i-th vertex of the j-th element
[in]ICELCELCell to cell pointer: ICELCEL(i,j) gives the element number that shares the face opposite the i-th vertex of the j-th element if 0 or > NELEM, that face is a boundary face
[in]ICELFACCell to face pointer: ICELFAC(i,j) gives the global face number of the face opposite the i-th vertex of the j-th element
[in]NOFVERTnof boundary faces
[in]NELEMnof boundary faces
[in]CORGCartesian coordinates of the meshpoints
[in]NDIMdimension of the space
[in]NPOINnof interior nodes in the mesh
[in,out]FACNORCartesian components of the normals to a face, multiplied by the face area
[in]NFACEnof faces in the mesh
[in]NBFACnof boundary faces
[in]NBINTnof inter-processor faces
