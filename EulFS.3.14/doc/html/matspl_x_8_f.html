MatSplitIX

Purpose

Compute the split matrices for the INcompressible preconditioned equations

Parameters
[in]IELEMthe current simplicial element
[in]NDIMthe dimension of the space
[in]NORDERthe order of the system to be solved
[in]VECNthe NDIM cartesian component of the inward face normal, scaled by its measure
[in]JacobianMatrixis the jacobian matrix
[in]LDJleading dimension of the jacobian matrix
[out]KMATthe jacobian matrix times the face normal
[out]KPOSis the "positive" part of the jacobian matrix
[out]KNEGis the "negative" part of the jacobian matrix
[out]VLEFTis the left eigenvector matrix of the jacobian matrix
[out]VRIGHTis the right eigenvector matrix of the jacobian matrix
[in]LDAis the leading dimension of the matrices KMAT, KPOS, KNEG, VLEFT, VRIGHT
[out]WRare the eigenvalues of the jacobian matrix
[out]LPOSare the positive eigenvalues of the jacobian matrix
[out]LNEGare the negative eigenvalues of the jacobian matrix
[in]EIGENDECO.TRUE. if the eigenvector decomposition of the matrix is needed, if .FALSE. only the KMAT matrix is computed
