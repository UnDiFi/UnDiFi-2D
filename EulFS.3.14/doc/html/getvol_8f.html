Parameters
[in]ICELNODCell to node pointer: ICELNOD(i,j) gives the global node number of the i-th vertex of the j-th element
[in]ICELFACCell to face pointer: ICELFAC(i,j} gives the global face number of the face opposite the i-th vertex of the ielem-th cell; If ICELFAC(i,ielem) > 0 the normal vector ( which is stored in VFACNOR(1:NDIM,IABS(ICELFAC(i,ielem))) ) points inside the ielem-th cell; outside if ICELFAC(i,ielem) < 0; the storage used for the face (edge) normals is as follows: a real array FACNOR(1:NDIM,1:NFACE) stores all the face(edge) normals of the mesh and an integer pointer: -NFACE <= ICELFAC(1:NOFVERT,1:NELEM) <= NFACE gives, with its absolute value, the normal opposite a given vertex of a given element
[in]NOFVERTnof boundary faces
[in]NELEMnof boundary faces
[in]FACNORCartesian components of the normals to a face, multiplied by the face area; Face normals : FACNOR(i,iface) gives the i-th cartesian component of the iface-th face. This is the vector normal to a triangular face (edge segment in 2D) scaled by the face area (edge length in 2D)
[in]NFACEnof faces in the mesh
[in]VCORGCartesian coordinates of the meshpoints
[in]NDIMdimension of the space
[in]NPOINnof interior nodes in the mesh
[out]VOLarea/volume of the simplicial elements (triangles,tetrahedra)
[in]TIMEthe time when the volumes are computed
