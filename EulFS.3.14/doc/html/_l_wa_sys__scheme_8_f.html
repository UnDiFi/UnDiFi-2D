Purpose

This routine computes the time-accurate LW system scheme on one tetrahedron;

it differs from the LDA version 1 system scheme since here the matrix multiplies from the left, i.e.

the distribution matrix for the time-accurate LW scheme reads:

Parameters
[in]MATRIXSPLITTERis the subroutine use to split the Jacobian matrix
[in]Wis the set of NORDER dependent variables in the NOFVERT vertices of the current cell
[out]NODRESare the signals sent to each of the NOFVERT vertices
[in,out]TSTEPis the elemental contribution to the (inverse of the) time step
[out]BETAthe NOFVERT distribution matrices
[out]STIFCthe elemental contribution to the implicit matrix (computed only if MATRIX_ASSEMBLY is true)
[in]NORDERis the order of the hyperbolic system being solved, it could be < than NDOF
[in]NDOFis the number of dofs in the gridpoints, also the leading dimension of W, STIFC, NODRES, TSTEP
[out]NOFVERTis number of vertices of the simplicial elements (= NDIM+1)
[in]VCNthe NDIM cartesian component of the inward face normal to all NOFVERT vertices, scaled by its measure
[in]NDIMis the dimension of the space
[in]DFGHDUthe NDIM cartesian components of the Jacobian matrix
[in]LDJis the leading dimension of DFGHDU
[out]CELRESthe elemental residual
[in]SOURCEthe elemental source term
[in]IELEMis the current simplicial element
[in]MATRIX_ASSEMBLYwhen set == .TRUE. the STIFC matrix will be assembled
