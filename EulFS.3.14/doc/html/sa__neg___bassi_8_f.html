SA_NAG_BASSI

Purpose

This routine discretizes the SA (standard) version of the Spalart-Allmaras turbulence model transport equation for the KINEMATIC viscosity to be used in the coupled solution approach for compressible flows

Nondimensionalization also leads to the redefinition of the and variables:

As far as the limiting of is concerned, we use:

which is referred to as "limiting (b)" in http://turbmodels.larc.nasa.gov/spalart.html

Parameters
[in]IELEMis the current simplicial element
[in]VCNthe NDIM cartesian component of the inward face normal to all NOFVERT vertices, scaled by its measure
[in]VCZis the set of NOFVAR dependent variables in the NOFVERT vertices of the current cell
[in]NDIMis the dimension of the space
[in]NOFVERTis number of vertices of the simplicial elements (= NDIM+1)
[in]NOFVARis the number of dofs in the gridpoints, also the leading dimension of VCZ, STIFEL, NODRES, TSTEP
[in]NTURBis the number of turbulent transport equations being solved
[out]NODRESare the signals sent to each of the NOFVERT vertices
[in,out]TSTEPis the elemental contribution to the (inverse of the) time step
[out]STIFELthe elemental contribution to the implicit matrix (computed only if MATRIX_ASSEMBLY is true)
[in]VOLUMEis the area/volume of the current cell
[in]MATRIX_ASSEMBLYwhen set == .TRUE. the STIFEL matrix will be assembled
[in]SCALARSCHEMEis the subroutine used to discretize the turbulent transport equation
[in]MATRIXSCHEMEis the subroutine used to discretize the turbulent transport coupled equations
[in]RWORKis a work array
[in]COMPRESSIBLEis logical flag, currently unused
[out]VISCLis the elemental (cell averaged) molecular laminar viscosity
[out]VISCTis the elemental (cell averaged) molecular turbulent viscosity
