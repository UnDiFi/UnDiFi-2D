the Motion solver: given the grid velocities at all boundaries, solves Laplace's equation to get the grid velocites at all interior gridpoints

Parameters
[in]MotionSolver
[in]PetscMatC
[in]NodalBcs
[in]NOFVERTnumber of vertices per element (=NDIM+1), since only triangles or tetrahedra are allowed)
[in]NPOINnof of interior (processor owned) meshpoints; global number of meshpoints in the uni-processor case
[in]NGHOSTnof of ghost meshpoints on the current processor; 0 the uni-processor case
[in]NPNODnof of periodic meshpoints on the current processor
[in]NELEMnof of processor owned elements (triangles/tetrahedra); global number of elements in the uni-processor case
[in]NBFACnof of processor owned boundary faces/edges; global number of boundary faces/edges in the uni-processor case.
[in]NFACEnumber of edges/faces in the mesh for the current processor (multi-processor case) or global number of edges/faces in the mesh (uni-processor case).
