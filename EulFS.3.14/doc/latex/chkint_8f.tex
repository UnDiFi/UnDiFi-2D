\begin{DoxyParagraph}{Purpose}

\end{DoxyParagraph}
Compute the integral of the conserved quantities over the entire computational domain $\Omega$\+: the routine has been written assuming that parameter vector is the dependent variable and therefore works for compressible flows only

The integral is written to {\ttfamily U\+N\+IT} = {\ttfamily I\+H\+S\+T4} 

In order to be consistent with the use of parameter vector as the set of dependent variables, the mean value of the conservative variables should be computed as follows\+: \begin{eqnarray*} <U>\,|\Omega| &=& \int_{\Omega} U \, \mathrm{d}V \\\ &=& \frac{1}{2} \, \sum_{e=1}^{NE} \int_{T_e} \left(\frac{\partial U}{\partial Z}\right) \, Z \, \mathrm{d}V = \frac{1}{2} \sum_{e=1}^{NE} \, \sum_{j \in e} \sum_{k \in e} \gamma_{jk}^e \left(\frac{\partial U}{\partial Z}\right)_{Z=Z_j} Z_k \end{eqnarray*} In deriving the equation above we have used the fact that the following relation holds\+: \[ U = \frac{1}{2} \left(\frac{\partial U}{\partial Z}\right) \, Z \] where both matrix $\partial U/\partial Z$ and vector $Z$ are linear functions of the independent variable $\mathbf{x}$. the coefficients $\gamma_{jk}^e$ that appear in the above equation are\+: \[ \gamma^e_{jk} = |T_e| \times \left\{ \begin{array}{cc} \frac{1}{20} & j \ne k \\ \frac{1}{10} & j = k \end{array} \right. \]
