{\bfseries{L\+D\+A\+S\+Y\+S\+\_\+\+S\+C\+H\+E\+ME}} 

\begin{DoxyParagraph}{Purpose}

\end{DoxyParagraph}
This routine computes the L\+DA version 1 system scheme on one tetrahedron;

it differs from the L\+DA version 2 system scheme since here the matrix $ K_i^+ $ multiplies $ \left( \sum_{j=1}^{d+1} K_j^+ \right)^{-1} $ from the right, i.\+e.

the distribution matrix for the L\+DA verion 1 scheme reads\+: \[ B_i = \left( \sum_{j=1}^{d+1} K_j^+ \right)^{-1} K_i^+ \]
